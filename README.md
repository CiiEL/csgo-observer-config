CS GO caster config
===================

# Installation
- Clone this repo into ../csgo/cfg/

# Usage
- From console run `exec caster/caster`
- Run `caster_help` to get more help


# Binds
- `Keypad *` CT base
- `Keypad -` T base
- `Keypad 9` Mid angle
- `Keypad 4` to `Keypad 6` Site A angles
- `Keypad 1` to `Keypad 3` Site B angles
- `Keypad 7` toggle hud
- `Keypad 8` toggle crosshair
